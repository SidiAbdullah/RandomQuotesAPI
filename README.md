# Random Quotes API

A simple ASP.NET Core API that returns random quotes (Arabic).

## Live Demo

- API: [https://randomquotesapi-4.onrender.com/api/Quotes](https://randomquotesapi-4.onrender.com/api/Quotes)  
- Interactive page: [https://randomquotesapi-4.onrender.com/Quotes.html](https://randomquotesapi-4.onrender.com/Quotes.html)

Click the button on the page to get a new random quote.
